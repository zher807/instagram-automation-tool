## 🧩 How to use the tools

### Using the parser
You can collect data by entering a link to a profile or a hashtag in the input field. Press the start button. The tool will save the information to a text file inside your folder. You can open these files with Notepad or Excel.

### Setting up mass actions
Choose the action you want to perform from the main menu. Load your list of targets from a text file. Adjust the delay settings to mimic human speed. A slower pace reduces the risk of account blocks. Always start with a small test run.

### Managing multiple accounts
You can add multiple logins within the settings menu. The software switches between these accounts automatically. This keeps your activity spread out and helps you maintain account health.
